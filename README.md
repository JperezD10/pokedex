# Ejecutar en desarrolo

1. Clonar el repo.
2. Ejecutar

```
yarn install
```

3. Tener Nest CLI 

```
npm i -g @nestjs/cli
```

4. Levantar mongo con docker

```
docker-compose up -d
```

## Stack

* NestJS
* MongoDB
* Docker
